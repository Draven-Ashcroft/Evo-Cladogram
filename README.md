## 🧬 What is this?

**Phylogenix** is an interactive evolutionary visualization platform designed to explore phylogenetic relationships, geological timelines, and taxonomic diversity.

The application allows users to navigate through major evolutionary lineages, inspect taxa, build and modify cladograms, attach scientific profiles, and visualize evolutionary history across hundreds of millions of years.

Built as a standalone educational and research-oriented tool, **Phylogenix** combines cladistics, paleontology, taxonomy, and scientific visualization into a single interactive experience.

## 🔬 Scientific Background

Phylogenetic cladograms are branching diagrams that represent hypotheses of evolutionary relationships among organisms.

By organizing taxa into nested lineages based on shared derived characteristics, scientists can reconstruct patterns of common ancestry and evolutionary divergence.

Phylogenix provides an interactive platform for exploring these relationships while integrating geological time, taxonomy, and biological information into a unified educational experience.
